HTML! Why it's useful

Have you ever been on the interenet? If you haven't get out of this century. If you have then you have seen the wonderful joys of HTML. All the websites, ALL OF THEM! are made using HTML. HTML is an acronym that stands for Hypertext Markup Language. HTML is written in programs such as Caret (this one is written in that). To view how it is written, right click on the page and click on "view souce info."

CSS! Why it's useful!

All websites designs are created using CSS. HTML is more the structure of the website, where CSS is the cosmetics. Regardless whenever you see a cool looking (or ugly) webpage it's thanks to CSS, or Cascading Style Sheets.

Tags, and how they work!

In HTML a tag tells the website what is happening. You can you a "div" tag to let the site know that a certain part is different from the rest. Or a "p" tag is used to create text. Some other examples are italics tags "em." Or Bold tags "strong." To create these tags all you need to do is substitute the quotation marks with eqaulity signs.

Atributes, and why they're useful!
