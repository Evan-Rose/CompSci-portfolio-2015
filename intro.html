WELCOME!

Welcome to computer science! In this class we spend the whole day coding, building/disassembling computers, and exploring OSes, like Linux. Computer science is an umbrella term, it describes many different things within that one term. We have mostly done work with software this week, designing websites and working with Linux. But we also touched on hardware when we dissasembled a computer, then reassembled it. Personally I liked the hardware exercise way more than the Linux one. This year we will also be doing other things, such as:

  2. Java Script
  3. AI (artifical intelligence)
